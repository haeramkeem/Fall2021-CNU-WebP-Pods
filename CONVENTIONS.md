# CONVENTIONS for PODs

## Branch Naming
* **main** : Main branch
* **develop** : Developing branch
* **Feature/...** : Feature branchs
* **Refactor/...** : Refactoring whole code
* **Fix/...** : Bug fix

## Commit Naming
* **feat: ...** : Add new features
* **fix: ...** : Bug fixes
* **docs: ...** : Modify documents
* **refactor: ...** : Refactor code
* **style: ...** : Reformat code
* **test: ...** : Add test codes
* **proto: ...** : Add prototype codes
